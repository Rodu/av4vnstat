This web interface allows you to visualize the data collected with vnstat - network traffic monitor - using a combination of different charts.

On top right you can see the date for the last update done on the dataset used in this front end.
You may want to run the Python script in order to update the dataset if you think it is is outdated by now.

For more information refer to the README and SETUP.txt files that you should have received with the project files.
